# 📚 Digital Contents Library Tool

웹페이지 내의 특정 콘텐츠 코드를 자동으로 인식하여 하이라이트하고, 마우스 오버 시 상세 정보를 미리 보여주는 크롬 확장 프로그램입니다. 나만의 라이브러리를 구축하고 관리하세요.

## 주요 기능

* **자동 하이라이트**: 웹페이지에서 설정한 정규식(Regex)에 맞는 코드를 자동으로 찾아 강조합니다.
* **퀵 프리뷰**: 하이라이트된 코드에 마우스를 올리면 콘텐츠의 이미지, 평점, 태그 정보를 즉시 확인합니다.
* **라이브러리 관리**: 소유 중인 콘텐츠를 목록화하여 관리하고, 텍스트로 내보낼 수 있습니다.
* **데이터 동기화**: 저장된 목록의 정보를 최신 메타데이터로 일괄 갱신합니다.
* **필터 및 검색**: 선호/불호 장르 설정, 제작사 필터링, 다크 모드를 지원합니다.
* **커스터마이징**: 미리보기 레이아웃, 하이라이트 색상을 자유롭게 설정할 수 있습니다.

## 🚀 설치 방법

이 확장 프로그램은 Chrome Web Store을 통해 설치 가능합니다.

## ⚙️ 설정 가이드 (중요)

확장 프로그램이 정상 작동하려면 사이트 규칙(Site Rules)을 등록해야 합니다.  

1.  확장 프로그램 아이콘을 클릭하여 팝업을 엽니다.
2.  `+` 버튼을 눌러 규칙 추가 폼을 엽니다.
3.  아래 예시를 참고하여 규칙을 입력하고 저장합니다.

| 필드 | 설명 | 예시 |
| :--- | :--- | :--- |
| **Rule Name** | 규칙 이름 (식별용) | `Example Site` |
| **Target API URL** | 정보를 가져올 API 주소 | `https://example.com/api/` |
| **Highlight Regex** | 찾을 코드의 정규식 패턴 | `(CODE-)\d{3,6}` |
| **Destination Code** | API 요청에 사용할 코드 포맷 | `CODE-{id}` |

> **Tip:** `{id}`는 정규식에서 추출된 숫자 부분으로 치환됩니다.

구체적인 엔드 포인트 API는 반드시 다음과 같은 형식을 만족시켜야 합니다.  
  
`{api}=/product.json?workno={item_code}`

## ❓ 자주 묻는 질문 (FAQ)

**Q. 미리보기가 뜨지 않아요.**
A. Target API URL이 올바른지, 해당 사이트가 CORS를 차단하지 않는지 확인해주세요.

**Q. 하이라이트 색상을 바꾸고 싶어요.**
A. 설정 -> 하이라이트 메뉴에서 '중복(소유함)'과 '미중복' 색상을 각각 지정할 수 있습니다.

## 🔒 개인정보 처리방침

이 확장 프로그램은 **사용자의 데이터를 외부 서버로 전송하지 않습니다.** 모든 데이터(설정, 라이브러리 목록)는 사용자의 브라우저 로컬 스토리지(`chrome.storage.local`)에만 저장됩니다.
